# Planejamentos (`docs/plans`)

- Esta pasta contém apenas **`completed/`** como planejamentos **finalizados**. Trate **`docs/plans/completed/`** como **referência histórica**, não como plano ativo nem como fonte prioritária frente a [`HANDOFF_MASTER.md`](../HANDOFF_MASTER.md), [`CHECKPOINT_CHECKLIST.md`](../CHECKPOINT_CHECKLIST.md) ou [`NEXT_PHASE_OFFICIAL.md`](../NEXT_PHASE_OFFICIAL.md), salvo quando o contexto pedir explicitamente o arquivo histórico.
- **Planos vivos** ficam em **`docs/`** na raiz da documentação. **Bloco 1 (LoanRequest operacional):** [`../PLANEJAMENTO_BLOCO1_LOANREQUEST_OPERACIONAL.md`](../PLANEJAMENTO_BLOCO1_LOANREQUEST_OPERACIONAL.md) — ao **concluir** o bloco, mover esse arquivo para **`docs/plans/completed/`**.
